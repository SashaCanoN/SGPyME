# 💼 SGPyME — Soluciones pastel y dinámicas para tu negocio

![Logo de SGPyME](img/Logo.png)

**SGPyME** es una plataforma web diseñada para potenciar pequeñas y medianas empresas con herramientas digitales visualmente agradables, funcionales y accesibles.

---

## ✨ Características principales

- 📱 **Diseño responsive** para móvil y escritorio
- 🖼️ **Estética pastel** moderna y amigable
- 🧭 **Menú animado** con botón hamburguesa ❌
- 🧃 **Tarjetas interactivas** para productos y servicios
- 📝 **Formulario pastel accesible**
- 🧑‍🤝‍🧑 **Sección de equipo, clientes y contacto**
- 📞 **Botón fijo de WhatsApp** animado con pulso y brillo
- 📖 Guía de estilos integrada (`style-guide.html`)
